# Practical Hours Tracking System (Streamlit App)

A lightweight, user-friendly system for managing:
- Student class lists  
- Site requirements (CRUD)  
- Practical hour logging  
- Progress tracking  
- Summary/completion status  

Built entirely in **Streamlit**, with **SQLite** for local storage.
